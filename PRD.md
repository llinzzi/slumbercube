# PRD: SSD1322 智能闹钟

**版本:** v1.0 | **日期:** 2026-05-19 | **状态:** Draft

---

## 1. 产品概述

一款基于 ESP32-C3 + SSD1322 256x64 灰度 OLED 的桌面智能闹钟，支持 WiFi 自动校时、天气预报显示、HTTP 音频流播放，以及低功耗深度睡眠。支持昼夜模式自动切换，白天全功能显示，夜晚极简低亮度显示以不影响睡眠。

---

## 2. 目标用户

| 画像 | 场景 |
|------|------|
| 桌面用户 | 需要一个非手机的信息终端，随时看时间/天气/听网络电台 |
| 睡眠敏感用户 | 夜间时钟需要极低亮度，无 WiFi 干扰，无高频噪音 |
| 极客/开发者 | 自编译定制固件，享受 DIY 硬件乐趣 |

## 5. 用户交互流程

```
上电/唤醒
  │
  ▼
[初始化] GPIO → SSD1322(关) → 按钮 → 时区 → LVGL → UI(首帧)
  │
  ▼
[显示开启] 首帧已渲染到 GDDRAM → 开显示(防白闪)
  │
  ├── 日间模式 (6:00~22:00)
  │   ├── WiFi 连接 + NTP 校时 + 天气获取
  │   ├── 网络电台播放
  │   ├── 主循环 600s (10分钟)
  │   │   ├── 每 1s 刷新时钟
  │   │   ├── 每 15s 刷新电台信息
  │   │   └── 按钮按下 → 立即睡眠
  │   └── 超时 → 深度睡眠
  │
  └── 夜间模式 (22:00~6:00)
      ├── 跳过 WiFi/天气/音频
      ├── 极简 7 段数码管 + 网格抖动 + 最低对比度
      ├── 主循环 600s
      └── 超时 → 深度睡眠
```

## 6. 技术架构

```
app_main()
├── GPIO 早期初始化
├── ssd1322_init()          — SPI 驱动，显示保持关闭 (0xAE)
├── Button init (GPIO3)
├── wifi_set_timezone()     — 设置 TZ=CST-8
├── lvgl_adapter_init()     — LVGL 显示适配器，L8→I4 转换
├── ui_wrapper_init()       — 创建主界面首帧
├── ssd1322_display_on()    — 首帧渲染后开显示
├── WiFi STA + SNTP 同步
├── Weather fetch (AMAP)
├── Audio init (NS4168 I2S)
├── Audio play URL (HTTP 流)
└── Main loop (600s, 1s tick)
    ├── Button poll
    ├── ICY station name polling
    └── → deep sleep
```

### 关键设计决策

| 决策 | 理由 |
|------|------|
| GPIO hold 防白闪 | 深度睡眠唤醒时 RST 保持低电平，防止 SSD1322 复位产生短时白屏 |
| 首帧渲染后开显示 | 确保 GDDRAM 有内容后再开显示 (0xAF)，彻底消除白闪 |
| L8→I4 转换 | LVGL 原生 L8(256 灰阶) → 4bit I4 适配 SSD1322 硬件 |
| 10 分钟活动周期 | 平衡功能可用性和功耗 |
| 按钮直连深睡 | 单键设计降低交互复杂度 |
| 夜间 Kconfig 时段 | 固定时段而非环境光检测，简单可靠 |

## 8. 已知限制

1. **Flash 容量矛盾** — README 声称 4MB，sdkconfig 配置为 2MB，需确认实际硬件
2. **WiFi 无配网 UI** — 凭据硬编码在 Kconfig，更换网络需重编译
3. **天气无缓存** — 每次唤醒重新获取，WiFi 失败则无天气显示
4. **无 OTA** — 固件更新需 USB 串口烧录
5. **单一交互** — 仅一个物理按钮，无法菜单导航
6. **FM116C 未启用** — 硬件预留的 LED 驱动无固件支持
7. **仅支持中文字体** — weather 和 station 字体为中文特定
8. **无持久化运行时配置** — 所有配置仅编译时确定

## 10. 成功指标（目标）

- ⏱ 网络校时成功 > 99% 唤醒次数
- 🌤 天气获取成功 > 95% 唤醒次数
- 🔊 音频流播放成功率 > 90%（受网络影响）
- 🌙 夜间模式白亮度不高于 1/16 正常亮度
- 🔋 单次唤醒 10 分钟消耗 < 电池容量的合理占比（需测量）
- 🚫 零白闪（100% 无视觉闪烁启动）
